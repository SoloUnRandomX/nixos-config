# NixOS Config
Configuracion de NixOS personal para mi Chromebook ^^
| Distro | Window Manager | Terminal | Topbar |
| ------ | -------------- | -------- | ------ |
| NixOS | Niri | Alacritty | Noctaria Shell |

## Como Instalar?
### NixOS
```
git clone https://github.com/SoloUnRandomX/nixos-config
cd nixos-config
chmod +x nixos.sh
./nixos.sh
```
### No-NixOS (Ubuntu, Arch, Fedora, etc.)
Por ahora no tengo soporte para estas distros ya que nunca las he probado junto a Nix, pero en un futuro planeo hacer un script para estas distros.
